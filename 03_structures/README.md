# Structural Files for gamma-Ga2O3

This directory contains all structural files related to gamma-Ga2O3, including structural configurations for Ga2, Ga3, and Ga4 sites.

## Key File Descriptions

**Representative structures selected from Ratcliff et al.**
- **`Ga2O3_113_1051.xyz`** - Structure with Ga3 site
- **`Ga2O3_113_1177.xyz`** - Structure with Ga4 site
- **`Ga2O3_113_1211.xyz`** - Structure with Ga2 site

**Structures optimized from the above configurations**
- **`Ga2.vasp`** - Structural configuration files for Ga2 site
- **`Ga3.vasp`** - Structural configuration files for Ga3 site
- **`Ga4.vasp`** - Structural configuration files for Ga4 site
